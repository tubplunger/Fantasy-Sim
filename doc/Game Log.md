# Fantasy-Sim

This is a Fantasy Simulation built entirely within Unity and using 3D assets to make the world. The goal of this game is to create a world using storytelling, fantasy worldbuilding, and gameplay to create something that reacts to the Player's presence and actions that they take.

Factions can evolve politically and quests emerge dynamically over time as the world progresses forward. NPCs remember the actions the the player takes which can change relationships over time and stories can arise naturally from these systems.
